# 小型超市管理系统

命令形式  小型超市管理系统 CUI界面

### 功能需求

- 有效记录商品信息
- 具备查询商品日、月销售数量
- 分为老板用户和店员用户
- 店员用户只有销售商品，查询商品的权限

### 系统设计

##### 功能模块：

- 商品入库出库记录
- 用户管理

##### 数据库设计：

- 用户信息(mb_user):用户ID(uid, 主键，大于100)， 用户名(uname, 不重复)， 密码(upass)， 手机号(phone)
- 商品信息(mb_goods):商品ID(gid, 主键)， 商品名称(gname, 不重复)， 商品成本价(cost_price)，商品售价(selling_price),商品剩余数量(counts)
